# Hand Gesture Recognition Interface

**Duration:** Sep 2025 – Oct 2025  
**Tech Stack:** Python | OpenCV | NumPy | Mediapipe

---

## Overview

This project is a **real-time hand gesture recognition system** that allows users to control media playback (play, pause, volume, next/previous track) using hand gestures.  

It uses **OpenCV** for video capture, **Mediapipe** for hand landmark detection, and **keyboard** library to simulate media control keypresses.

---

## Features

- **Real-time Hand Tracking** with landmark visualization.
- **Gesture-based Media Controls**:
  - **1 Finger** → Volume Down
  - **2 Fingers** → Volume Up
  - **3 Fingers** → Next Track
  - **4 Fingers** → Previous Track
  - **5 Fingers** → Play/Pause
- **Stable Gesture Detection** to reduce false actions.
- **Dual Modes**:
  - Hand Tracking
  - Gesture-based Media Player Control
